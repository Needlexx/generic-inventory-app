Generate PWA Icons

This tool creates the two PNG files your app needs for home-screen / PWA installs (especially on iOS). It draws from favicon.svg by default, or you can upload your own image.

icon-192.png (192×192)
icon-512.png (512×512)

Next steps

  1. Click Download Both PNGs — you'll get icon-192.png and icon-512.png
  2. Move both files into your project's icons/ folder, overwriting the old ones
  3. Open a terminal in the project and run:
    git add icons/ && git commit -m "Update PWA icons" && git push
  4. On each device, uninstall the old PWA and reinstall so the new icon shows
